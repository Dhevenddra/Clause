04 · The trust coreClause.

Zero hallucination by construction

Trust is enforced by code, not promised by prompt.

1 · Extract

Gemma must quote the policy character-for-character for every decision point. One structured call — no chains, no agents.

→

2 · Verify

validation.py: normalized exact-substring match against the source, mapped back to a character span. No fuzzy matching, no embeddings — a quote either exists or it doesn't.

→

3 · Stamp

Pass → ✓ verified + clause highlight.
Fail → ✗ rejected in the open, excluded from the decision basis. Never silently dropped.

8/8citations verified across live demo scenarios
11/11validation edge cases pass — paraphrase & fabrication rejected

05 · On AMDClause.

Open-weight Gemma, AMD-ready by design

  • Gemma 4 31B IT (open weights) is the only adjudication model — served via the Fireworks AI API on a dedicated deployment.
  • Same app runs Gemma on an AMD Instinct MI300X with vLLM on ROCm — 192 GB HBM3 runs it on a single card.
  • Switching is one environment variable. No code changes.
  • Open weights + self-hosting = the on-prem, air-gapped deployment story regulated insurers actually require.
